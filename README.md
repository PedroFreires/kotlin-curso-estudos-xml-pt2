# Kotlin Curso - Estudos com XML (Parte 2)

Este repositório contém os projetos e códigos desenvolvidos, e replicados por mim, durante o curso de kotlin com foco em desenvolvimento Android utilizando XML. Ele serve como registro do meu aprendizado prático, onde replico exemplos, exploro conceitos e realizo exercícios do curso.

## 📚 Conteúdo do Repositório

Cada pasta representa uma seção ou tópico específico abordado no curso. Nelas, você encontrará exemplos práticos de:

- Criação de Fragments
- Ciclo de vida das Activities
- Intents e navegação entre telas
- Manipulação de elementos de UI com XML
- Tratamento de eventos de clique
- Utilização de Toolbars e Menus
- Componentes do Material Design
- Práticas com `ViewBinding`

## 🚀 Tecnologias Utilizadas

- **Kotlin** (linguagem principal)
- **Android Studio**
- **ViewBinding**
- **XML para Layouts**
- **Material Design Components**

## 🧠 Objetivo

Meu objetivo é reforçar os conceitos aprendidos no curso por meio da prática constante, manter o versionamento do meu progresso e tornar meu aprendizado público — facilitando futuras revisões e também ajudando outros iniciantes que desejam aprender Kotlin e desenvolvimento Android.
